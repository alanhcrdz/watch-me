# Installation

## 1. Clone this repo

```git clone https://github.com/alanhcrdz/watch-me```
## 2. Go to project folder
```cd watch-me```

## 3. Install dependencies

```yarn```</br>

## 4. Start Json server
```yarn server```

## 5. Start application
```yarn dev```
